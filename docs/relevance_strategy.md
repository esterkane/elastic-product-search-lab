# Relevance Strategy

This document will track how product ranking is designed, tested, and improved.

Initial focus areas:

- Query understanding for exact, partial, and category-driven intent.
- Field weighting across title, brand, category, attributes, and description.
- Filter and facet behavior for structured product discovery.
- Hybrid retrieval experiments that combine lexical and vector signals.
- Regression testing with offline judgments before changing rank logic.
