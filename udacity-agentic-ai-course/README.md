# Udacity Agentic AI Course

This repository is organized as a working archive for Udacity Agentic AI coursework. It is structured to support four course projects over time. Projects 01 and 02 are represented now; projects 03 and 04 should be added later using the same documentation pattern.

The repository is intentionally documentation-heavy. Each project should include source files or notebooks, captured outputs when useful, architecture notes, lessons learned, do/don't sections, troubleshooting notes, and difficult quiz questions.

## Repository Layout

```text
docs/       Course-level explanations and reference notes
notes/      Topic notes that cut across projects
projects/   One folder per course project
```

## Security Rules

- Do not commit real API keys, tokens, credentials, private endpoints, or local machine paths.
- Do not commit `.env` files. Use documented environment variable names and safe placeholder values only.
- Review notebooks before committing. Outputs can contain prompts, model responses, stack traces, or copied secrets.
- Keep generated files only when they are evidence for a course requirement or useful for review.

## Evidence Policy

Do not claim a project was tested unless the repository contains the command, output, timestamp, or artifact that supports the claim. When evidence is missing, write `TODO: add test evidence`.

## How To Add A Project

1. Create `projects/NN-project-name/`.
2. Add `README.md`, `docs/architecture.md`, `docs/lessons-learned.md`, `docs/dos-and-donts.md`, `docs/troubleshooting.md`, and `docs/quiz.md`.
3. Add source files, notebooks, and outputs only after checking for secrets.
4. Update `docs/course-overview.md` and this README.
